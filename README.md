# TravelVerse - Developer Guide

Welcome to the TravelVerse repository! This is a simple MERN stack travel blog application. 

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB

## How to run locally

### 1. Backend Setup
```bash
cd backend
npm install
npm run dev
```
Make sure you have a local MongoDB instance running or update the `.env` file with your connection string.

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
